A **pattern** in a Regular Expression (RegEx) is a **sequence of characters** that defines a **search format**. Patterns allow us to match, find, or replace text based on rules instead of exact strings.

Patterns are made using:
- **Literals** â†’ direct characters (`/cat/` matches "cat").
- **Metacharacters** â†’ special symbols with meaning (`.`, `*`, `+`, `?`, `^`, `$`).
- **Character classes** â†’ sets or ranges (`[a-z]`, `\d`).
- **Quantifiers** â†’ number of repetitions (`+`, `*`, `{n,m}`).
- **Groups** â†’ parentheses for sub-patterns (`(abc)`).
