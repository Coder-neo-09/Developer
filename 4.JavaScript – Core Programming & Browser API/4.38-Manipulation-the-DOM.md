## âœ… Summary

| Task              | Method                                   |
| ----------------- | ---------------------------------------- |
| Select elements   | `getElementById`, `querySelector`, etc.  |

## ğŸ§  Best Practices

1. âœ… Use `textContent` instead of `innerHTML` (safer, prevents XSS).
2. âœ… Minimize **reflows** by manipulating DOM in batches (use `documentFragment`).
3. âœ… Use `classList` for styling instead of `.style`.
4. âœ… Keep DOM manipulation separate from **business logic**.
5. âœ… Always clean up unused nodes (remove event listeners).
