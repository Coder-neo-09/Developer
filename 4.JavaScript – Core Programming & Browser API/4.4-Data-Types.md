## ðŸ“Œ 5. Type Conversion

### **5.1 Implicit Conversion (Type Coercion)**

```js
"5" + 2; // "52" (string)
"5" - 2; // 3   (number)
```

### **5.2 Explicit Conversion**

```js
Number("42");     // 42
String(42);       // "42"
Boolean(0);       // false
```
