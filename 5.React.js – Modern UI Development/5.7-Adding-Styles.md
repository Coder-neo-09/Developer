## 3Ô∏è‚É£ CSS Modules  

- CSS scoped to a specific component.

```css
  background-color: purple;
  color: white;
  padding: 10px;
}
```

```jsx

```

‚úÖ Prevents **class name conflicts**.

## 4Ô∏è‚É£ Styled Components (CSS-in-JS)  

- Uses the `styled-components` library.  
- Write CSS directly inside JavaScript.  

```jsx

  background-color: orange;
  color: white;
  padding: 10px;
  border: none;
  border-radius: 5px;
`;

  return <Button>Click Me</Button>;
}
```

‚úÖ Great for **component-specific styles**.

‚úÖ Allows **custom styles via props**.

## üß† Best Practices

* Follow a **consistent naming convention**.
